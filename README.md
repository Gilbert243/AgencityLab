# AgencityLab

**AgencityLab** is a scientific Python framework dedicated to the study and quantification of Agencity — a proposed observable designed to measure a system’s ability to actively structure its own evolution.
Agencity can be analyzed at a single scale or across multiple scales using the agencity spectrum b(τ).
