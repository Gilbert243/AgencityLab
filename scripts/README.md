# AgencityLab Scripts

This directory contains portable helper scripts for the AgencityLab project.

Files:
- `build_docs.sh` / `build_docs.bat` — build project documentation
- `run_tests.sh` / `run_tests.bat` — run the test suite
- `release.sh` / `release.bat` — build distribution artifacts
- `clean.sh` / `clean.bat` — remove temporary build outputs

The scripts are intentionally simple and rely on standard Python tooling
so they can be adapted easily on Linux, macOS and Windows.
