## Overview

AgencityLab is a scientific framework for the theory of Agencity.  
It provides tools for preprocessing, canonical computation, analysis,
visualization and reproducible experimentation.

## Core ideas

- canonical signal: $u(\xi)$
- reduced activation: $X^*$
- reduced activity: $A^*$
- memory: $M$
- organization: $O$
- agencement: $\beta$
- observable Agencity: $b$
