# Principles

## Principle 0 — Existence

For every admissible input signal, the Agencity pipeline must be computable
without requiring access to the internal mechanism of the system.

## Principle 1 — Objectivity

The observable depends on measurable quantities only.

## Principle 2 — Saturation

The structured agencement $\beta$ stays within $(-1, 1)$.

## Principle 3 — Non-conservation

Agencity is not a conserved quantity.

## Principle 4 — Temporal asymmetry

The construction uses causal windows and finite differences, so the observable
is not symmetric under time reversal.

## Principle 5 — Substrate independence

The framework can be applied to physical, informational, biological or
abstract dynamical systems, provided a coherent coordinate $\xi$ and signal
$u(\xi)$ can be defined.
